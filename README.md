## Using Local Models with Ollama

1. Install Ollama  
   https://ollama.com

2. Run a model:

```bash
ollama run llama3.2
```

3. Ensure Ollama runs at:

```
http://localhost:11434
```

AI Hub auto-connects if Ollama is active.
